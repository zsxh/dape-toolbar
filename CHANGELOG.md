# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.2] - 2026-03-17

### Added
- Optional `PREDICATE` parameter to `dape-toolbar-buttons` for conditional button visibility
- `disconnect` button to the default button list

### Changed
- Enhanced `:type` customization widget with descriptive tags for better UX
- Improved `fit-window-to-buffer` call with `window-live-p` check and error handling

## [0.0.1] - Initial Release

### Added
- Basic toolbar functionality with continue, step-over, step-in, step-out, restart, and quit buttons
- Mouse click and keyboard support
- Customizable button height
- Integration with dape info buffer window groups
